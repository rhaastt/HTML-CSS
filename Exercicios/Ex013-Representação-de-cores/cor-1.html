cores em css

exemplo de representação cores

As Cores podem ser representadas por seus nomes
ou por seu codigo em hexadecimal que é uma base numérica bastante utilizada

ex por nome = color: blue

BLUE

ex de hexadecimal = #0000ff que é a cor azul

AZUL

ou pelo mais simples que é a escala RGB
e o visual code permite isso.


Tambem é possivel modificar a transparencia pela tabela de cores do VISUALCODE
