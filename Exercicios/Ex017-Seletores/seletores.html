Utilizando Seletores

Usando ID

O ID é associado a um elemento do html através da tag ID="nomedoid" e para utilizar no css
ele é chamado com o #nomedoid.

Não é aconselhado usar o mesmo ID dentro do mesmo documento, na regra do w3c isso esta errado.

Usando class

As classes são usados igual o ID, porem utilizando a tag class="nomedaclass"
e é chamada no css com o .nomedaclass

Alem de poder utilizar em multiplas partes do documento, tambem é possivel adicionar mais de uma palavra na mesma classe

Diferenças

ID E CLASS

O ID não é aconselhado á usar multiplos com o mesmo nome, isso é função da CLASS
Tambem é possivel utilizar CLASS e ID no mesmo elemento
