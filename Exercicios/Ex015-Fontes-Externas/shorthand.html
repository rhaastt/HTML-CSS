ShortHand

Para simplificar todas essas configurações é usado um "shorthand" que é um font:

exemplo: { font:italic bolder 3em 'work sans', sans-serif }

A ordem dos atributos de uma shorthand em CSS é importante. No caso da propriedade font, devemos informar a oderm: :

  1. font-style
  2. font-variant
  3. font-weight
  4. font-size/line-height
  5. font-family
