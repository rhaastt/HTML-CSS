Categoria de fontes

Fontes Serifadas

As serifas eram usadas na época das prensas manuais, eram aplicas em grandes blocos de textos.
As serifas tem a capacidade de guiar nossos olhos durante a leitura das palavras.Atualmente são usadas em titulos

Exemplos de fontes serifadas

  • Bree Serif
  • EB Garamond
  • Noto Serif (Usada pelo google)
  • Bitter

Fontes sem Serifas

Mais conhecida como fontes sans-serif (em francês) que são fontes mais limpas e mais usadas na web
são otimos para exebição em telas por produzir uma sensação de limpeza, clareza e organização

Exemplos de fontes sem Serifa

  • Open Sans
  • Titillium web
  • Oswald
  • Montserrat

Fontes Monoespaçadas

É uma categoria que deriva das duas fontes anteriores, por isso existem fonte Monoespaçadas com ou sem serifa.
A caracterista principal é o espaçamento das letras entre si, elas possuem o mesmo.

A principal vantagem é tornar a leitura mais facil e que requerem que você reproduza.
Por isso é bem usada para linhas de codigo, e assim elas tambem ficam conhecidas como fonte terminal ou fonte console

Exemplos de fontes Monoespaçadas

  • Souce Code Pro
  • Roboto Mono
  • IBM Plex Mono
  • Ubuntu Mono

Fontes Script

São fontes handwriting, são aquelas que tentam reproduzir a escrita humana
São fontes bem cursivas e usadas em palavras ou partes de textos especificos por sua dificuldade de leitura. São consideradas fontes que falam.

Exemplos de Fontes Script

  • Great Vibes
  • Dancing Script
  • Pacifico
  • Reenie Beanie

Fontes Display

São fontes que fogem do padrão com bastante efeito visual, efeitadas e curiosas.
Tambem são chamadas de fontes comemorativas, e algumas delas nem são letras, possuindo animais, simbolos, objetos, pessoas e personagens de quadrinhos.

É uma fonte recomendada para criar titulos em destaque que chamam atenção

  • Lobster
  • Luckiest guy
  • Bungee Outline
  • Press Start2P

Indicação de uso

Geralmente é declarado uma combinação segura de fontes, para que mesmo se não haver um
tipo de fonte no dispositivo de quem estiver acessando, vai ter uma de reserva.

Um site bom para averiguar isso é o CSS WEB SAFE-FONTS

As melhores fontes para paragrafos são sem serifa, elas facilitam a leitura do cliente/usuario
