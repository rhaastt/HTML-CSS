Semanticas das divs

No Html existe tags semanticas para dividir melhor o documento:

Header

É nada mais nada menos que o cabeçario da pagina, aonde geralmente esta o menu que tambem possui uma tag semantica para essa divisão dentro do cabeçario, e a tag é NAV

Main

É a tag aonde vai todo o conteudo do site e dentro dele possui diversas tags para melhorar a separação dos elementos.

Section

É uma tag para separar elementos dentro de uma determinada classe como:noticias, documentarios, artigos e etc.

Article

O article é para dividir elementos da mesma classe. Pensa em um jornal com varias noticias e cada noticia é um article.

Aside

É uma tag que fica dentro da article. É usado para um elemento relativo, como o criador daquele artigo ou informações especiais.

Footer

A tag footer é utilizada para separar o rodapé da pagina
